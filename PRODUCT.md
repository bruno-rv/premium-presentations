# Product

## Register

brand

## Users

Presentation authors and technical speakers use this project to build polished
HTML slide decks that can be opened directly, bundled for sharing, and presented
with live controls. The audience is usually watching projected or screen-shared
slides, so clarity at room distance matters more than app-like density.

## Product Purpose

Premium Presentations provides reusable HTML deck templates, shared theme
tokens, controls, annotations, presenter tools, and bundling scripts. Success
means a deck feels deliberate at thumbnail size, readable in a live room, and
robust enough to present without local setup friction.

## Brand Personality

Editorial, technical, high-polish.

## Anti-references

Avoid generic SaaS deck templates, text-heavy conference slides, decorative
chrome that competes with the idea, nested card grids, low-contrast muted text,
and AI-default palettes that make unrelated decks feel interchangeable.

## Design Principles

- One slide, one dominant read.
- Theme choices should change mood without breaking the shared layout grammar.
- Controls and presentation chrome should support the speaker, then disappear
  for the audience.
- Visual systems should be distinctive at contact-sheet scale.
- Proof objects, diagrams, and readable labels beat paragraphs.

## Accessibility & Inclusion

Aim for WCAG AA contrast for text and controls, visible focus states, keyboard
operability, reduced-motion support, and slide text large enough to survive
projection or screen sharing.
